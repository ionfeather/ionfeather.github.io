Cusdis评论系统测试

这个页面用于测试Cusdis评论系统是否正常工作。

测试步骤：

  1. 查看页面底部是否有评论框
  2. 检查评论框是否显示中文标签
  3. 尝试提交一条测试评论
  4. 检查评论是否显示

预期结果：

  • 评论框应该显示在页面底部
  • 所有标签应该是中文（昵称、邮箱、网站）
  • 可以成功提交评论
  • 评论会出现在下方

问题排查：

如果看不到评论系统：

  1. 检查浏览器控制台（F12 → Console）
  2. 检查网络请求（F12 → Network）
  3. 检查Cusdis App ID是否有效

注意：测试完成后可以删除此页面。
